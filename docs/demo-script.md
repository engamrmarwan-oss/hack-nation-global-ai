## Demo Flow

### [0:00] Opening Hook (15 seconds)
**Say:** "What if you could [do X] in [N seconds] instead of [N hours]?"
**Do:** Show the end result first — the "wow" moment.

---

### [0:15] Problem Statement (30 seconds)
**Say:** "Right now, [target user] has to [painful manual process]. It's slow, error-prone, and [consequence]."
**Show:** [Screenshot or quick live demo of the problem — optional]

---

### [0:45] Solution Demo — Happy Path (90 seconds)

**Step 1:**
- **Say:** "I'll [action]..."
- **Do:** [Exact UI action — e.g., paste input into field, click button]
- **Show:** assets/sample-inputs/[filename]

**Step 2:**
- **Say:** "The system now [processes/analyzes/generates]..."
- **Do:** [Watch loading, point out what's happening]

**Step 3:**
- **Say:** "And here's the result..."
- **Do:** [Show output]
- **Show:** assets/sample-outputs/[filename]

### [2:15] Key Technical Highlight (30 seconds)
**Say:** "The interesting part is [how AI component works]. We're using [model/technique] to [achieve X]."
